# Bandit Wargame Automation Bot

这是一个自动完成 OverTheWire Bandit 闯关游戏的 Python 程序。该程序可以自动通过 SSH 连接到不同关卡的服务器，执行必要的命令来获取密码信息，并自动进入下一关。

## 功能特性

- 自动 SSH 连接管理
- 命令执行和结果解析
- 密码管理和存储
- 关卡进度追踪
- 详细的日志记录

## 使用方法

1. 安装依赖：

```bash
pip install paramiko
```

2. 检查 `config.py` 配置。

3. 运行程序

```bash
python run.py
```
